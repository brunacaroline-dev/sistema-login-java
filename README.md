# Sistema de Login em Java

Projeto desenvolvido para praticar lógica de programação em Java.

## Funcionalidades
- Validação de usuário e senha
- Limite de 3 tentativas
- Bloqueio após tentativas inválidas

## Tecnologias
- Java

## Como executar
1. Compile:
   javac Login.java

2. Execute:
   java Login

## Objetivo
Projeto voltado para aprendizado em Análise e Desenvolvimento de Sistemas.
